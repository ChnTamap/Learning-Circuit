# 一、项目

## 1.光控灯

LED、电阻、电位器、光敏电阻、三极管的简单应用

要点

* 发光二极管和镇流电阻
* 电位器和分压电路
* 三极管，小电流控制大电流

[示例](./项目/1.光控灯/光控灯.md)

## 2.直流电源

二极管、变压器、电容的简单应用

要点

* 半桥整流、全桥整流器
* 交流电和变压器
* 了解电容的充放电、滤波

## 3.双闪-多谐振荡器

三极管、电容的简单应用

初步接触振荡电路

* 例程
